# El-Ahorcado-UIDE
# Juego del Ahorcado (Python)

## Objetivo
Desarrollar el juego del ahorcado aplicando lógica de programación, estructuras condicionales y repetitivas.

## Contenido del repositorio
- **/src/main.py**: código fuente del juego
- **/diagramas/**: diagramas de flujo del sistema (general, juego y validaciones)

## Funcionalidades implementadas (avance)
- Selección aleatoria de palabra
- Ingreso de letras por teclado
- Validación de entrada
- Control de intentos
- Condición de victoria y derrota

## Cómo ejecutar (local)
1. Instalar Python 3
2. Abrir una terminal en la carpeta del proyecto
3. Ejecutar:
   ```bash
   python src/main.py
